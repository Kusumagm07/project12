AgriTech — Crop Demand Forecasting Platform

Demo-ready summary — 5 minute presentation

This demo shows a lightweight crop demand forecasting platform with: predictive models, market insights, and a recommendation engine. The frontend is a simple static UI served by a Flask backend with a few API endpoints in backend/src/app.py.

Key Pages

  • Login: `index.html` — demo login (pre-filled credentials available for quick demo).
  • Welcome: `welcome.html` — summary dashboard and entry to Dashboard.
  • Functionalities: `functionalities.html` — complete list of backend endpoints and features (recommended to open this last).

Quick Demo Steps (2 minutes)

  1. Start backend: cd backend then python -m flask --app src/app.py run
  2. Open http://localhost:5000 in a browser (use private/incognito to avoid stale localStorage).
  3. Use demo credentials: mahesha@gmail.com / m@123. Click Sign In.
  4. Open Functionalities to show backend API and features.
